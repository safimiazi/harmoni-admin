# Prince LMS Dashboard App

The **Prince LMS Dashboard App** is an admin panel built with **Next.js** to manage content and users for the **Prince LMS Android App**. Admins can upload health awareness videos, manage subscriptions, and control access to ensure only authorized users can view the content on the mobile app.

---

## 📌 Purpose

This dashboard is **not a public web app**, but rather a control panel for managing the Android-based Learning Management System (LMS). It is intended for admin use only.

---

## 🚀 Features

- 🎥 Upload & manage health-related awareness videos
- 👥 Admin login with secure authentication
- 🔐 Role-based access (only admins can use the dashboard)
- 📱 Controls content shown in Android LMS app
- 📊 Dashboard analytics (coming soon)
