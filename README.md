# Prince LMS Dashboard App

The **Prince LMS Dashboard App** is an admin panel built with **Next.js** to manage content and users for the **Prince LMS Android App**. Admins can upload health awareness videos, manage subscriptions, and control access to ensure only authorized users can view the content on the mobile app.
